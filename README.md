# Configs
All my linux config files in one place.
These configs are subject to change as my preferences change.
Feel free to do whatever you want with these files

## Usage
You can make symbolic links to the files and folders you want in appropriate
places, or integrate this in your own management system. Do it whichever way it works for you.
