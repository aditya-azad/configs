# Configs

All my config files in one place

## Usage

- Use the `install.sh` file in the scripts directory to install the configs on linux based system
- Use batch file on windows system to copy neovim or emacs configs to and from the system
  - **Be sure to modify the script to put your own user name**. This is not the best solution to the problem but it works
- Or copy the files manually to appropriate places
