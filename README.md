# Configs

Arch linux bootstraper and config files

## Disclaimer

- These configs are **subject to change** as my preferences change
- Do read the scripts before running them as **some of them delete already existing files**
- A few scripts **may require additional setup** to work on your system

## Usage

- Use the `install.sh` file in the scripts directory to install the configs on arch based linux system
- Use batch file on windows system to copy neovim configs to and from the system
- Or copy the files in `configs` folder manually to appropriate places
